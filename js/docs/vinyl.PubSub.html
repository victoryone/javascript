vinyl. PubSub

발행/구독 객체: 상태변화를 관찰하는 옵저버(핸들러)를 등록하여, 상태변화가 있을 때마다 옵저버를 발행(실행) 하도록 하는 객체이다.

Example

// 옵저버 등록
vinyl.PubSub.on('customevent', function() {
	 alert('안녕하세요');
});

// 등록된 옵저버 실행
vinyl.PubSub.trigger('customevent');
